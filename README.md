非稳态工作模式：非稳态（Astable，也叫自激多谐振荡器）是 555 定时器的经典工作模式之一，电路无需外部触发信号，自动、持续地在两个稳定状态（高电平和低电平）之间来回切换，输出连续的方波 / 脉冲信号，因此也叫 “无稳态” 模式（没有固定的稳定状态，只有两个暂稳态）。单稳态模式需要外部触发信号才能产生单次脉冲，双稳态模式需要外部信号翻转状态，只有非稳态模式可以独立、持续地生成时钟 / 脉冲信号，是 “自激振荡” 的核心应用。电路结构极简：仅需 2 个电阻 + 1 个电容 + 555 芯片，就能实现稳定的脉冲发生，成本极低、可靠性高。选择它的核心原因，结构极简、成本极低、无需外部触发、频率 / 占空比灵活可调，是生成连续脉冲信号的最优方案之一，广泛应用于时钟、PWM、LED 驱动等场景。

用端口实现模块间接口！！！之所以被称为端口是因为在电子学中，我们通常认为数据是通过被称为“端口”的小通道来传输的，一个GPIO或者其他什么信号端口、控制端口，都可以称为端口，我们可以给它打上标签，比如说叫做A0、A1......
如果我们直接把它们接到LED上，他能正常点亮LED吗？需要看数据手册看看源端或灌电流的输出电流是多少（当芯片引脚输出低电平（逻辑 0）时，其内部连接到地（GND）的晶体管导通，外部电流通过负载（如 LED、电阻）流入该引脚，最终流向芯片内部的地。这个被 "灌入" 芯片的电流，就叫灌电流。）
灌电流：
      电流方向：外部电源 → 负载 → 芯片引脚 → 内部地（GND）
      发生条件：引脚输出低电平（0V）
      本质：芯片引脚吸收外部电流
      常见能力：单片机 / MCU 单引脚灌电流通常为 8~25mA
以 5V 单片机驱动 LED 为例：
      灌电流接法（常用）
      LED 阳极 → 限流电阻 → VCC（+5V）
      LED 阴极 → 单片机 IO 引脚
      当 IO 输出低电平（0），电流灌入 IO，LED 亮
      计算公式：Isink​=RVCC​−VLED​−VOL​

​设计注意事项
绝对不能超过最大值：每个引脚都有最大灌电流限制（如 25mA），超过会烧毁芯片。
总电流限制：整个芯片所有引脚的总灌电流也有限制（如 150mA），需整体核算。
优势：灌电流能力通常强于拉电流，是驱动 LED、继电器的首选方式。

关于电阻的材质问题：通常来说金属膜电阻会更贵，精度也更高。碳质电阻最便宜，碳膜电阻会好一些，陶瓷电阻则比较特别。如果你想要高精度，就选金属膜电阻。如果你只是想便宜又好用，就用碳质或碳膜电阻。公差的选择：如果我们要做非常先进的计时电路，为了实现极高的计时精度，会选择低至0.01%的，常见的是5%。关于功率，使用标准的四分之一瓦的电阻。
在挑选LED时，我们可以把正向电压设置得非常低，因为我们想让这东西尽快亮起来（如1.8~2.2）

关于布局：当你拥有所有这些机械元件时，必须确保它们放在正确的位置。如果有外壳的话，为了可制造性，会不会出现碰撞或干扰与机械部位或其他东西之间。
关于布线：芯片的摆放位置或方向会使布线变得非常困难或更容易。
通常将去耦电容或旁路电容尽可能靠近电源引脚，这样它才能发挥作用，把那些噪音旁路掉，通常你会看到我们把IC电容放在正上方往右偏一点，我们之所以这样做，是因为如果我们把它直接放在上面，布线会有点困难。
我们将在整个板子上进行地线填充，这意味着我们不需要担心接地的问题，通过填充操作，我们就能直接得到地线，填充操作会把所有空白区域都填满，所有有接地的地方都会连接上。
我们将手动布线VCC电源，有两层、四层、六层、八层......的电路板存在，在那些电路板中，你可能有一整层专门用于3.3V电压或者其他电压，比如1.2V，你会进行三维布线来获取电源，所以实际上，电源线你根本不需要在二维层面上布线，有一整个电源平面。但现在，我们没有足够的平面可用，只有顶层和底层来传输信号，所以我们会将这些平面用作信号层，也就是信号平面。
如果数据总线或某种总线上有信号，信号特性相似且都需同步到达，你需要让这些信号线彼此平行布设并尽量保持他们的走线长度也就是每根导线的长度一致，这样当他们以接近光速沿着导线传播时（通常是光速的0.7倍，它们就能同时到达，从而避免所谓的信号偏移。如果你不想一条线上的噪声耦合到另一条线上，你就不能让这些线平行布设，而是要垂直布设。另外，当制作双层板时，你会选择方向和走线方式来在顶层布设你的导线，顶层全部水平布设，底层全部垂直布设。我们会在顶部和底部都布设地平面，地平面有助于电路板的散热，也有助于减少电路板上的杂散电容，有助于处理电路板边缘的电场和磁场，这样它们就有了一个返回的路径，它也为电荷在顶层走线中移动提供了电流路径，如果其下方有地平面，就能提供一个电容耦合的场所，这样场线就不会散布到外部空间了。你可能会因为各种原因没有接地层，而且它还会形成一个所谓的屏蔽+，这样它就能限制住电场和磁场。
应该让所有走线都尽可能宽，这样它们就能承载尽可能多的电流，具有最小的阻抗。要让电源走线宽于信号走线。因此，如果你有信号走线和信号电路，你谈论的是纳安、微安、毫安级别的电流流过电路，那些走线不需要很宽，但电源轨或电源线，你确实希望它们非常宽，因为他们会给所有这些集成电路供电。
当你在进行电路布局和布线时，在这些相当简单的印刷电路板上，目标是你的信号走线尽量做到8mil到10mil,12mil甚至更好。至于电源线尽量做到12mil到15mil，甚至20mil，30mil,40mil.
当你在大型总线上有信号传输时，你可以让所有信号同时沿着总线传输，它们必须有一个回流路径，然后你只有一个很小的接地回流路径，这就是为什么在设计良好的总线上，你会看到信号路径而且你会看到回流路径紧随其后或紧邻其旁。这样你就会有信号路径，然后是回流路径；接着是信号路径，再是回流路径。因此，如果设计得当，许多大型总线电缆都是这样工作的。
