用3-8线解码器，这些解码器很多时候都用于内存逻辑电路。我们这里有IC芯片、HC系列芯片、CMOS集成电路芯片，这意味着它们可以在3——6V的电压范围内工作所以我们需要某个电源供电，我们打算用电池供电，选择电池座以便安装，使用三节1.5V电池串联起来，空载时就能得到大约4.5V的电压，一旦接入负载电压会下降随着电路消耗电流增多，这个电压也会跟着变化，电池没电时电压大幅下降。

非稳态工作模式：非稳态（Astable，也叫自激多谐振荡器）是 555 定时器的经典工作模式之一，电路无需外部触发信号，自动、持续地在两个稳定状态（高电平和低电平）之间来回切换，输出连续的方波 / 脉冲信号，因此也叫 “无稳态” 模式（没有固定的稳定状态，只有两个暂稳态）。单稳态模式需要外部触发信号才能产生单次脉冲，双稳态模式需要外部信号翻转状态，只有非稳态模式可以独立、持续地生成时钟 / 脉冲信号，是 “自激振荡” 的核心应用。电路结构极简：仅需 2 个电阻 + 1 个电容 + 555 芯片，就能实现稳定的脉冲发生，成本极低、可靠性高。选择它的核心原因，结构极简、成本极低、无需外部触发、频率 / 占空比灵活可调，是生成连续脉冲信号的最优方案之一，广泛应用于时钟、PWM、LED 驱动等场景。

用端口实现模块间接口！！！之所以被称为端口是因为在电子学中，我们通常认为数据是通过被称为“端口”的小通道来传输的，一个GPIO或者其他什么信号端口、控制端口，都可以称为端口，我们可以给它打上标签，比如说叫做A0、A1......
如果我们直接把它们接到LED上，他能正常点亮LED吗？需要看数据手册看看源端或灌电流的输出电流是多少（当芯片引脚输出低电平（逻辑 0）时，其内部连接到地（GND）的晶体管导通，外部电流通过负载（如 LED、电阻）流入该引脚，最终流向芯片内部的地。这个被 "灌入" 芯片的电流，就叫灌电流。）
灌电流：
      电流方向：外部电源 → 负载 → 芯片引脚 → 内部地（GND）
      发生条件：引脚输出低电平（0V）
      本质：芯片引脚吸收外部电流
      常见能力：单片机 / MCU 单引脚灌电流通常为 8~25mA
以 5V 单片机驱动 LED 为例：
      灌电流接法（常用）
      LED 阳极 → 限流电阻 → VCC（+5V）
      LED 阴极 → 单片机 IO 引脚
      当 IO 输出低电平（0），电流灌入 IO，LED 亮
      计算公式：Isink​=RVCC​−VLED​−VOL​

​设计注意事项
绝对不能超过最大值：每个引脚都有最大灌电流限制（如 25mA），超过会烧毁芯片。
总电流限制：整个芯片所有引脚的总灌电流也有限制（如 150mA），需整体核算。
优势：灌电流能力通常强于拉电流，是驱动 LED、继电器的首选方式。
