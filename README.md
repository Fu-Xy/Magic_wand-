时钟源选择555定时器，其工作频率有公式（取决于两个电阻一个电容），与其使用固定电阻不如使用电位器，这样就能调整频率了；
将这个频率输入到一个四位计数器里，这样我们就可以将信号输入到四位计数器，让它开始计数，让每个代码都对应点亮一个LED，要对这个四位的计数进行解码，需要一个4到16位的解码器；
用3-8线解码器，这些解码器很多时候都用于内存逻辑电路。我们这里有IC芯片、HC系列芯片、CMOS集成电路芯片，这意味着它们可以在3——6V的电压范围内工作所以我们需要某个电源供电，我们打算用电池供电，选择电池座以便安装，使用三节1.5V电池串联起来，空载时就能得到大约4.5V的电压，一旦接入负载电压会下降随着电路消耗电流增多，这个电压也会跟着变化，电池没电时电压大幅下降。
如果是数字电路，就不能随便接一个电池就完事了还不加任何稳压

之所以要在电池底座旁边加一个电容器，是因为一旦我们打开电源，并且这个设备通电后，VCC将由这些电池供电，当我们使用这些电池时，电池的电压会发生变化，随着电流从它们那里被抽取，这是因为每个电池都有内部阻抗，所以需要加一个电容器来平稳电压。当我们需要电荷时，它会先从电容器中获取，然后再从电池中获取，因为电容器的阻抗会更低，所以要放一个大电解电容。目前VCC只是一个抽象的概念，是一个网络名称。软件知道这是电源，但没有给它分配电压或其他参数，因为没有仿真，对软件来说仅仅是文本。
二、555定时器：使用555定时器作为系统的“心脏”，来生成一个时钟信号（生成时钟信号的方法有很多种，比如用可编程的集成电路IC、微控制器、晶体管振荡器、场效应管振荡器等等）。我们更看重封装而不是具体的零件型号，除非确实需要使用某个非常独特的IC或元件才会关注其精确的零件型号。旁路电容C和两个电位器。C是指CMOS版本

非稳态工作模式：非稳态（Astable，也叫自激多谐振荡器）是 555 定时器的经典工作模式之一，电路无需外部触发信号，自动、持续地在两个稳定状态（高电平和低电平）之间来回切换，输出连续的方波 / 脉冲信号，因此也叫 “无稳态” 模式（没有固定的稳定状态，只有两个暂稳态）。单稳态模式需要外部触发信号才能产生单次脉冲，双稳态模式需要外部信号翻转状态，只有非稳态模式可以独立、持续地生成时钟 / 脉冲信号，是 “自激振荡” 的核心应用。电路结构极简：仅需 2 个电阻 + 1 个电容 + 555 芯片，就能实现稳定的脉冲发生，成本极低、可靠性高。选择它的核心原因，结构极简、成本极低、无需外部触发、频率 / 占空比灵活可调，是生成连续脉冲信号的最优方案之一，广泛应用于时钟、PWM、LED 驱动等场景。
